## Sobre o Projeto
Auto Login é uma interface gráfica desenvolvida em Python utilizando Tkinter para demonstrar o preenchimento automático de campos sensíveis, como usuário e senha, dentro de um ambiente controlado. O projeto apresenta como funcionalidades internas podem manipular estados de entrada em uma GUI com segurança e clareza.

## Importância Técnica do Projeto
Este projeto destaca conhecimentos essenciais para aplicações desktop e automação:

### **Manipulação de Interfaces Gráficas com Tkinter**
O uso de Tkinter evidencia domínio na construção de janelas, rótulos, campos de entrada e botões, mostrando capacidade de criar interfaces responsivas e funcionais.

### **Gerenciamento de Estados Internos**
A utilização de variáveis dinâmicas para armazenar e atualizar o conteúdo dos campos demonstra domínio de `StringVar` e do ciclo de vida dos componentes, fundamentais para GUIs bem estruturadas.

### **Encapsulamento Funcional**
A lógica de preenchimento automático é abstraída em funções internas, reforçando boas práticas de modularidade e organização em projetos Python.

### **Automação e Interação Direta com o Usuário**
O comportamento de auto-preenchimento mostra habilidade para criar fluxos automatizados dentro de aplicações — uma técnica utilizada em sistemas corporativos e ferramentas internas.
